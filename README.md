# Library-Management-System-FullStack
A full-stack Library Management System designed to automate and simplify the management of library operations such as book inventory, user management, borrowing/returning books, and fine tracking. The system provides separate roles for Admin and Users, ensuring secure and efficient access control.
How to Run:
Install MongoDB 
Clone Project
At Terminal:
Start Backend: mvn spring-boot:run
Start Database: mongosh 
Start frontend: npm run dev
